# Tests

Test harness and reference fixtures for the Audit Resource Tracker.

## Structure

- `golden_files/` — reference workbooks used for parity testing. The
  Sprint 2 refactor's output is compared against these files cell by
  cell. Any deviation is treated as a regression unless explicitly
  documented as an intended improvement.

## Current Contents

- `golden_files/v4_reference.xlsx` — 11-tab workbook generated by
  `reference/build_v4.py`. Established as the Sprint 2 fidelity target.
  Contains: Holidays & Skeleton, Audit Setup, and 9 Resource tabs
  (PM, Asst PM, Auditors 1-4, AM, QC, RE).
